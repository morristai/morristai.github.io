{{- /* Project override of DoIt's render-image hook. Rendering strategy (deliberately diverges from theme): 1. In-page: always a fixed-width 600 JPG thumbnail. No srcset, no WebP. Hugo preserves ICC profiles on jpg→jpg resize but strips them on WebP output (known Hugo limitation), which makes wide- gamut source photos (P3 phone JPGs) render dim in-page while the lightbox-opened original looks correct. Using a JPG thumbnail makes in-page colours match the clicked original. 2. Hover: CSS scale(1.02) in assets/css/_custom.scss (.post-image). 3. Click: lightgallery opens the *original* resource at full size. points to $resource.RelPermalink (original), not the 600-wide thumbnail, so lightbox shows full-resolution. Remote images and data URIs bypass the resource pipeline and render as a plain inline img (matches theme fallback behaviour). */ -}} {{- $lightgallery := .Page.Params.lightgallery | default site.Params.Page.lightgallery | default false -}} {{- $isUrlRemote := urls.Parse .Destination | partial "function/isUrlRemote.html" -}} {{- $isInline := strings.HasPrefix .Destination "data:image" -}} {{- /* Resolve the source as a page resource when possible. Remote URLs and data URIs fall through to a plain inline img below. */ -}} {{- $resource := "" -}} {{- if not (or $isUrlRemote $isInline) -}} {{- $resource = (.Page.Resources.Get .Destination) | default (resources.Get .Destination) -}} {{- end -}} {{- /* Render helpers: $thumbSrc (in-page 600-wide jpg), $fullSrc (original full-size). For SVGs or when resize is not applicable, $thumbSrc falls back to the original so we don't break display. */ -}} {{- $thumbSrc := .Destination -}} {{- $fullSrc := .Destination -}} {{- $thumbW := "" -}} {{- $thumbH := "" -}} {{- if $resource -}} {{- $fullSrc = $resource.RelPermalink -}} {{- $isSvg := eq $resource.MediaType.SubType "svg" -}} {{- if $isSvg -}} {{- $thumbSrc = $resource.RelPermalink -}} {{- else -}} {{- /* Only downscale if the source is wider than our target. Upscaling small screenshots would blur them. */ -}} {{- if gt $resource.Width 600 -}} {{- $thumb := $resource.Resize "600x jpg q90" -}} {{- $thumbSrc = $thumb.RelPermalink -}} {{- $thumbW = $thumb.Width -}} {{- $thumbH = $thumb.Height -}} {{- else -}} {{- $thumbSrc = $resource.RelPermalink -}} {{- $thumbW = $resource.Width -}} {{- $thumbH = $resource.Height -}} {{- end -}} {{- end -}} {{- end -}} {{- $alt := .Text -}} {{- $title := .Title -}} {{- if $title -}} {{- else -}} {{- if $lightgallery -}} {{- end -}} {{- if $lightgallery -}} {{- end -}} {{- end -}}
